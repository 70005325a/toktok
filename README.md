# 💬 톡톡 메신저 (카카오톡 스타일 · LAN 실시간 메신저)

같은 네트워크(WiFi/사무실 망)의 **여러 PC가 실시간으로 연결**되는 카카오톡 스타일 메신저입니다.
**Python 표준 라이브러리만** 사용하므로 추가 설치가 필요 없습니다. (이 PC에 Python 3.11 설치됨)

## ✨ 기능

- 🙂 프로필 — 아바타 이모지/사진 업로드 + 닉네임 + 상태메시지
- 👥 친구 탭(접속자 목록) · 💬 채팅 탭(방 목록·안읽음 배지)
- 🗨️ 그룹 채팅 + 1:1 채팅 · 카톡 스타일 말풍선·아바타·날짜 구분선
- 🔢 읽음 표시(안 읽은 인원 수) · ✍️ 입력 중 표시
- ↩️ 답장 · 🗑️ 삭제 · 📋 복사 · 🔍 검색
- 😊 이모지 · 🖼️ 사진 전송 · 🌙 다크모드 · 📲 PWA 설치
- 💾 **대화 영구 저장(SQLite)** — 서버를 꺼도 채팅방·대화 유지
- 🔒 **입장 비밀번호** — 외부 공개 시 코드를 아는 사람만 입장
- 😀 **메시지 반응(공감)** · ✏️ **메시지 수정** · 🗑️ 삭제 · ↩️ 답장(인용·점프)
- 📎 **파일 전송** · 🎤 **음성 메시지(녹음)** · 🔗 **링크 자동 인식** · @ **멘션**
- 🔔 **알림 + 알림음**(다른 방/백그라운드/멘션 시) · ↵ **여러 줄 입력**(Shift+Enter)
- ⬇️ **맨 아래로/새 메시지** 버튼 · ⋮ **채팅방 이름 변경·참여자 보기**

- 📌 **공지 고정** · ↪️ **메시지 전달** · 🚫 **사용자 차단** · 👀 **"여기까지 읽음" 구분선**
- 🟢 **마지막 접속 시간**(1:1) · 🔕 **방별 알림 끄기** · 🖼️ **이미지 확대 뷰어** · 📥 **드래그&드롭/붙여넣기 업로드** · 🎨 **채팅방 배경 변경**

> 💡 사용 팁: 말풍선을 **탭**하면 답장/전달/복사/공지고정/수정/차단/삭제/공감 메뉴가 열립니다.
> 입력창에서 **Shift+Enter**는 줄바꿈, **Enter**는 전송. 이미지를 끌어다 놓거나 붙여넣기로도 보낼 수 있어요.
> 채팅방 우측 상단 **⋮** 메뉴에서 이름 변경·참여자·알림 끄기·배경 변경을 할 수 있습니다.

## 🟢 어떻게 켜나요? (배치파일 더블클릭)

| 파일 | 용도 |
|------|------|
| **`톡톡_시작.bat`** ⭐ | **원클릭 외부 공개** — 서버+터널 자동 실행, 공개 주소를 크게 표시 + 클립보드 자동 복사, 비밀번호 설정 |
| **`톡톡_앱.bat`** | **내 PC에서 앱처럼 열기** — 서버 자동 시작 후, 주소창 없는 독립 창(앱 모드)으로 메신저 실행 |
| `톡톡_바로가기.html` | 바탕화면에 두고 더블클릭하는 **바로가기 페이지**(내 PC 열기 / 외부 주소 붙여넣어 열기) |
| **`고정주소_설정.bat`** | **고정(영구) 주소 1회 설정** — ngrok 가입·토큰·도메인 입력 (한 번만) |
| **`톡톡_고정주소.bat`** ⭐ | **항상 같은 주소로 공개** — 매번 안 바뀜. 설정 후 이것만 더블클릭 |
| `톡톡_열기.url` | (설정 후 자동 생성) 더블클릭하면 고정 주소로 바로 접속하는 바로가기 |
| `톡톡_종료.bat` | 서버·터널 강제 종료(정리) |
| `내네트워크_시작.bat` | 같은 WiFi(LAN)에서만 사용 |

채팅방 우측 상단 **⋮** 메뉴에서 **🗑️ 채팅방 삭제**(모든 참여자에게서 제거)도 가능합니다.

### 🔗 항상 같은 주소로 (고정 주소)
- 내 PC를 켜둔 채 고정 주소: `고정주소_설정.bat`(1회) → `톡톡_고정주소.bat` — 자세히는 `고정주소_안내.md`
- **PC를 꺼도 24시간 접속(클라우드 배포)**: **`배포_안내.md`** 참고 — `https://이름.onrender.com` 영구 주소

> 가장 쉬운 방법: **`톡톡_시작.bat` 더블클릭** → (선택)비밀번호 입력 → 표시되는 `https://...trycloudflare.com` 주소를 친구에게 전달(이미 복사됨). 이 창을 켜 둔 동안 접속 가능, **Enter**로 종료.

## 🚀 실행 (이 PC = 서버)

```powershell
cd "C:\Users\70005\OneDrive\Desktop\업무정리\클로드 스킬\messenger-web"
python server.py
```

실행하면 콘솔에 접속 주소가 표시됩니다:
- 이 PC:        **http://localhost:4173**
- 다른 PC/폰:   **http://192.168.0.60:4173**  ← 같은 WiFi에서 브라우저로 접속

> 포트 변경:  `$env:PORT=8080; python server.py`

## 🌍 인터넷 외부 공개 (다른 WiFi/외부에서도 접속) — 무료

`외부공개_시작.bat` 를 **더블클릭**하면 끝입니다.
서버 + Cloudflare 무료 터널이 함께 켜지고, 콘솔에 공개 주소가 표시됩니다:

```
https://xxxx-xxxx-xxxx.trycloudflare.com   ← 이 주소를 상대방에게 전달
```

- 상대방은 **인터넷 어디서나**(다른 WiFi, LTE 등) 이 https 주소로 접속하면 바로 대화 가능
- 가입·결제 불필요(완전 무료). 동봉된 `cloudflared.exe` 사용
- ⚠️ **이 PC가 켜져 있어야** 동작합니다(이 PC가 서버). 창을 닫으면 끊깁니다.
- ⚠️ 무료 터널은 **켤 때마다 주소가 바뀝니다.** → 고정 주소는 `고정주소_안내.md` 참고
- 🔒 실행 시 **입장 비밀번호**를 정할 수 있습니다(엔터로 건너뛰면 공개). 코드를 아는 사람만 입장됩니다.
- ⚠️ 공개 주소를 아는 사람은 누구나 입장할 수 있습니다 → 입장 비밀번호가 필요하면 요청하세요.

> 수동 실행:
> ```powershell
> # 터미널 1
> python server.py
> # 터미널 2
> .\cloudflared.exe tunnel --url http://localhost:4173
> ```

## 🔗 같은 네트워크(LAN)에서 연결하는 법

**파일을 복사할 필요가 없습니다.** 한 대만 서버를 켜고, 나머지는 브라우저로 접속만 합니다.

```
[이 PC] python server.py 실행  ← 서버 (계속 켜져 있어야 함)
   │
   ├─ 이 PC 브라우저   → http://localhost:4173
   └─ 다른 PC 브라우저 → http://192.168.0.60:4173   (같은 WiFi)
```

1. **이 PC**에서 `python server.py` 실행 (서버를 켜 둠)
2. **다른 PC**(같은 WiFi)에서 브라우저를 열고 `http://192.168.0.60:4173` 접속
3. 각자 닉네임을 정하고 시작 → 친구 탭에 서로 보이고, 같은 채팅방에서 실시간 대화 ✅

### ⚠️ 처음 연결이 안 될 때 (Windows 방화벽)
다른 PC에서 접속이 안 되면, 이 PC의 방화벽이 막는 경우입니다. 관리자 PowerShell에서:
```powershell
New-NetFirewallRule -DisplayName "TokTok Messenger" -Direction Inbound -Protocol TCP -LocalPort 4173 -Action Allow
```
(또는 처음 `python server.py` 실행 시 뜨는 "액세스 허용" 창에서 허용)

## 💾 대화 저장 / ⏻ 서버 특성

- 메시지는 **서버 메모리**에 저장됩니다 → 서버(이 PC)를 켜 둔 동안 모든 PC가 같은 대화를 공유.
- **서버를 끄면(프로그램 종료) 대화가 사라집니다.** 영구 저장이 필요하면 파일/DB 저장을 추가하면 됩니다(요청 시 구현).
- 인터넷 어디서나(다른 WiFi/외부망) 연결하려면 클라우드 배포가 필요합니다.
