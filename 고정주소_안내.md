# 🔗 고정(영구) 주소 만들기 — 항상 같은 주소로 접속

무료 ngrok 고정 도메인을 쓰면, **켤 때마다 바뀌지 않는 같은 주소**로 접속할 수 있어요.
어려운 설정은 다 만들어 뒀고, **딱 1회 가입+입력**만 하면 됩니다.

## 1단계 — 1회 설정  (`고정주소_설정.bat` 더블클릭)
실행하면 안내가 뜨고, 아래 3가지를 차례로 합니다(모두 무료):

1. **가입**: https://dashboard.ngrok.com/signup
2. **토큰 복사**: https://dashboard.ngrok.com/get-started/your-authtoken
   → "Your Authtoken" 의 긴 문자열 복사 → `고정주소_설정.bat` 창에 붙여넣기(마우스 우클릭) + Enter
3. **무료 도메인 발급**: https://dashboard.ngrok.com/domains
   → **+ New Domain** 클릭 → `예: happy-cat-1234.ngrok-free.app` 생성 → 그 주소를 창에 입력 + Enter

끝나면 자동으로:
- 도메인이 저장되고 (`ngrok_domain.txt`)
- **`톡톡_열기.url`** (더블클릭하면 그 고정 주소로 바로 여는 바로가기)이 생깁니다.

## 2단계 — 실행  (`톡톡_고정주소.bat` 더블클릭)
- 서버 + 고정주소 터널이 함께 켜지고, 화면에 **항상 같은 주소**가 표시(+클립보드 복사)됩니다.
- 이 창을 켜 둔 동안 친구가 그 주소로 접속 가능. 끝내려면 **Enter**.

> 그 주소(또는 `톡톡_열기.url`)를 친구에게 한 번만 알려주면, 다음부턴 **매번 같은 주소**라 다시 안 보내도 됩니다.

## 매번 새로 안 해도 되나요?
- **설정(1단계)은 평생 1회**면 됩니다.
- 이후엔 **`톡톡_고정주소.bat` 더블클릭**만 하면 같은 주소로 열립니다.

## ⚠️ 알아둘 점
- **내 PC가 켜져 있고 이 창이 떠 있어야** 친구가 접속됩니다(내 PC가 서버라서).
  PC를 꺼도 항상 접속되게 하려면 "클라우드 배포"가 필요해요(원하면 도와드립니다).
- ngrok 무료는 첫 접속 시 "방문 확인" 페이지가 한 번 뜰 수 있습니다(계속 누르면 됨).

## (대안) 내 도메인이 있다면 — Cloudflare 고정 터널
도메인을 소유하고 Cloudflare에 등록돼 있으면:
```powershell
.\cloudflared.exe tunnel login
.\cloudflared.exe tunnel create toktok
.\cloudflared.exe tunnel route dns toktok chat.내도메인.com
# 서버 실행 중인 상태에서:
.\cloudflared.exe tunnel run --url http://localhost:4173 toktok
```
→ `https://chat.내도메인.com` 이 고정 주소가 됩니다.
